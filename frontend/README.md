# Frontend Dashboard (Next.js)

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Features

- Enterprise layout (Header + Sidebar)
- Feature-based architecture
- Typed mock data
- Dashboard table with loading, error, and empty states
- Clean import aliases (`@/*`)

## Architecture

- src-based structure
- reusable layout components
- feature-driven folders

## Impact

- Designed scalable frontend architecture suitable for international teams
- Implemented UX-safe states for real-world data fetching
- Ready for backend integration (REST / GraphQL)

## Notes

This project is part of a monorepo including a NestJS backend.
